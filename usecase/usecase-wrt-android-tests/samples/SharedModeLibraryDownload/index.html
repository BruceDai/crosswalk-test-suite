Precondition

  1. Setup a web server and make sure the test device can connect it
  2. Make sure the test device has not install XWalkRuntimeLib.apk

Test Steps

  1. Build a webapp with command:
    $ python make_apk.py --package=[package] --name=[name] --app-url=https://crosswalk-project.org/ --xwalk-apk-url=[http://host/XWalkRuntimeLib.apk] --mode=shared --project-dir=[dirName]
  2. Open the AndroidManifest.xml from directory [dirname] and check if there has define a meta-data element with the name "xwalk_apk_url" inside the application tag
  3. Install and launch it
  4. Click "Get Crosswalk", check the screen
  5. Click "Install", then check the screen
  6. Click "Done", then check the screen
  7. Define an element with the key "xwalk_apk_url" in res/manifest.json and set your server address to the value, just like:
    "xwalk_apk_url": "http://host/XWalkRuntimeLib.apk"
    then build a webapp with below command:
    $ python make_apk.py --package=[package] --manifest=[path to res/manifest.json] --mode=shared
  8. Repeat the steps 2, 3, 4, 5, 6
  9. Build a webapp with command:
    $ python make_apk.py --package=[package] --manifest=[path to res/manifest.json] --mode=shared --project-dir=[dirName] --project-only
  10. Define a meta-data element with the name "xwalk_apk_url" inside the application tag in file AndroidManifest.xml, just like:
    <application android:name="org.xwalk.core.XWalkApplication">
    <meta-data android:name="xwalk_apk_url" android:value="http://host/XWalkRuntimeLib.apk" />
    ...
    </application>
    Then rebuild this app again
  11. Repeat the steps 3, 4, 5, 6
