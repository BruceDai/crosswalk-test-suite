Precondition

  1. Please refer to README file

Test Steps

  1. If org.test.foo exists in usecase-apptools-tests/tools/ directory, remove it
    Then run below commands under usecase-apptools-tests/tools/ directory:
    $ node %crosswalk-app% create org.test.foo --platforms=windows --windows-crosswalk=
    $ cd org.test.foo
    $ node %crosswalk-app% build
  2. Install foo app on windows
  3. Right-click app icon and open properties dialog
  4. Launch the app on Windows, right-click on the app
  5. Click "Inspect Element"
  6. $ node %crosswalk-app% build release
  7. Install the app on Windows
  8. Right-click app icon and open properties dialog
  9. Launch the app on Windows, right-click on the app
  10. Uninstall foo app on windows

Purpose:

Verifies foo app can run successfully

Expected Result:

  1. Create/Build org.test.foo successfully
  2. Install the foo app successfully
  3. The "--enable-inspect" show up at the end of the Target entry
  4. Launch the foo app successfully and there will be a popup menu "Inspect Element" on right-click
  5. Open the inspection page successfully
  6. Build org.test.foo successfully
  7. Install the foo app successfully
  8. The "--enable-inspect" doesn't show up at the end of the Target entry
  9. Launch the foo app successfully and the popup menu will not be shown on right-click
  10. Unistall the foo app successfully
