Preconditions:

  1. Please refer to README file

Test steps:

  1. If com.example.spacedodge exists in apptools-sampleapp-tests/tools/ directory, remove it
    Then run below commands under apptools-sampleapp-tests/tools/ directory:
    $ crosswalk-app create com.example.spacedodge
    $ cd com.example.spacedodge
    $ git clone https://github.com/crosswalk-project/crosswalk-samples.git
    $ mv crosswalk-samples/space-dodge-game/base/* app/
    Update "xwalk_version" to "xwalk_app_version" in app/manifest.json file
    Add '"xwalk_package_id": "com.example.spacedodge", "xwalk_target_platforms": ["ios"]' to app/manifest.json file
    $ crosswalk-app build
  2. Connect your iOS device to your iMac. Open iTunes->"Your Device"->Apps
    Double click spacedodge app. Click "Install" button of spacedodge app on iTunes
  3. Click "Apply" button at the lower right corner of iTunes. Check icon of spacedodge app in your iOS device
  4. Launch spacedodge app in your iOS device
  5. Check the homepage of spacedodge app
  6. Close spacedodge app via clicking the home button, and kill spacedodge app from Task Manager
  7. Long press the icon of spacedodge app to uninstall it

Expected Output:

  1. Create/Build com.example.spacedodge successfully
  2. Install the spacedodge app successfully
  3. Find the spacedodge app successfully
  4. Launch the spacedodge app successfully
  5. You can see spacedodge UI on the device
  6. Stop the spacedodge app successfully
  7. Uninstall the spacedodge app successfully
