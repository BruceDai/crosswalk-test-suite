Preconditions:

  1. Please refer to README file

Test steps:

  1. If com.example.gallery exists in apptools-sampleapp-tests/tools/ directory, remove it
    Then run below commands under apptools-sampleapp-tests/tools/ directory:
    On Android:
        Shared Mode:
            $ crosswalk-app create com.example.gallery --android-shared
        Embedded Mode:
            $ crosswalk-app create com.example.gallery
        $ cd com.example.gallery
        $ git clone https://github.com/blueimp/Gallery.git
        $ mv Gallery/* app/
        $ crosswalk-app build
    On Windows:
        $ node %crosswalk-app% create com.example.gallery
        $ cd com.example.gallery
        $ git clone https://github.com/blueimp/Gallery.git
        $ xcopy /s /e /i /y Gallery\* app\
        $ node %crosswalk-app% build
  2. Install gallery app on your android device which is "arm" architecture
  3. Check icon of gallery app in your android device
  4. Launch gallery app in your android device
  5. Check the homepage of gallery app
  6. Uninstall the app when it is running
  7. Install and launch the gallery app again. Close gallery app via clicking the home button, and kill gallery app from Task Manager
  8. Long press the icon of gallery app to uninstall it

Expected Output:

  1. Create/Build com.example.gallery successfully
  2. Install the gallery app successfully
  3. Find the gallery app successfully
  4. Launch the gallery app successfully
  5. You can see Gallery UI on the device
  6. Uninstall the gallery app successfully when the app running
  7. Stop the gallery app successfully
  8. Uninstall the gallery app successfully
