Preconditions:

  1. Please refer to README file

Test steps:

  1. If com.example.spacedodge exists in apptools-sampleapp-tests/tools/ directory, remove it
    Then run below commands under apptools-sampleapp-tests/tools/ directory:
    On Linux or OS X host:
        Shared Mode:
            $ crosswalk-app create com.example.spacedodge --android-shared
        Embedded Mode:
            $ crosswalk-app create com.example.spacedodge
        Lite Mode:
            $ crosswalk-app create com.example.spacedodge --android-lite
        $ cd com.example.spacedodge
        $ git clone https://github.com/crosswalk-project/crosswalk-samples.git
        $ mv crosswalk-samples/space-dodge-game/base/* app/
        Update "xwalk_version" to "xwalk_app_version" in app/manifest.json file
        Add '"xwalk_package_id": "com.example.spacedodge", "xwalk_target_platforms": "android"' to app/manifest.json file
        $ crosswalk-app build
    On Windows host:
        Shared Mode:
            $ node %crosswalk-app% create com.example.spacedodge --android-shared
        Embedded Mode:
            $ node %crosswalk-app% create com.example.spacedodge
        Lite Mode:
            $ node %crosswalk-app% create com.example.spacedodge --android-lite
        $ cd com.example.spacedodge
        $ git clone https://github.com/crosswalk-project/crosswalk-samples.git
        $ xcopy /s /e /i /y crosswalk-samples\space-dodge-game\base\* app\
        Update "xwalk_version" to "xwalk_app_version" in app\manifest.json file
        Add '"xwalk_package_id": "com.example.spacedodge", "xwalk_target_platforms": "android"' to app\manifest.json file
        $ node %crosswalk-app% build
  2. Shared Mode:
        Install shared mode spacedodge app on your android device which is "arm"     Install shared mode spacedodge app on your android device which is "x86" architecture
    Embedded Mode or Lite Mode:
        Install arm spacedodge app on your android device which is "arm" architecture     Install x86 spacedodge app on your android device which is "x86" architecture
  3. Check icon of spacedodge app in your android device
  4. Launch spacedodge app in your android device
  5. Check the homepage of spacedodge app
  6. Uninstall the app when it is running
  7. Install and launch the spacedodge app again. Close spacedodge app via clicking the home button, and kill spacedodge app from Task Manager
  8. Long press the icon of spacedodge app to uninstall it

Expected Output:

  1. Create/Build com.example.spacedodge successfully
  2. Install the spacedodge app successfully
  3. Find the spacedodge app successfully
  4. Launch the spacedodge app successfully
  5. You can see spacedodge UI on the device
  6. Uninstall the spacedodge app successfully when the app running
  7. Stop the spacedodge app successfully
  8. Uninstall the spacedodge app successfully
