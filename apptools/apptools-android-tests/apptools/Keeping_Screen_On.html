Test steps:

  1. If org.xwalk.test exists in apptools-android-tests/tools/ directory, remove it
    Then run below commands under apptools-android-tests/tools/ directory:
    On Android:
        $ ./crosswalk-app-tools/src/crosswalk-app create org.xwalk.test
        $ cd org.xwalk.test
        Set "xwalk_android_keep_screen_on" to "true" in app/manifest.json file
        $ ../crosswalk-app-tools/src/crosswalk-app build
    On Windows:
        $ node .\crosswalk-app-tools\src\crosswalk-app create org.xwalk.test
        $ cd org.xwalk.test
        Set "xwalk_android_keep_screen_on" to "true" in app\manifest.json file
        $ node ..\crosswalk-app-tools\src\crosswalk-app build
    Install the app on android device
    Set the device screen off as 15 seconds(Setting->Display->Sleep: 15 seconds)
    Launch the app on android device
    Check the screen

Expected Results:

  1. Web app keeps screen on status more than 15 seconds
