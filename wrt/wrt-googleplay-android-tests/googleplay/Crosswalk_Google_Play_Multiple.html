Pre-condition:

  1. Two test devices, one with x86 architecture, another one with arm architecture
  2. Make sure above two test devices have Goole Play installed, and be able to access Google Play
  3. An available google play developer account

Test steps:

  1. Sign in the Google Play Developer Console on browser and click the button "Add new application"
  2. Input a title as application name and click the button "Upload APK"
  3. Choose "ALPHA TESTING" or "BETA TESTING" item and click the button "Upload new APK to Beta", then upload the "test_multiple.apk" into the application
  4. Click the button "SWITCH TO ADVANCED MODE"
  5. Click the button "Upload new APK to Beta", then upload the "test_multiple/test_multiple.apk" into the same application
  6. Click the link "Manage list of testers", then add Google Groups or Google+ Communities info
  7. Fill in the required information in "Store Listing" and "Pricing & Distribution", then publish the application(Note: it will take some time to audit the application from google play)
  8. Click the link "Manage list of testers" and copy the link which should be shared with testers, then forward to the link to become a tester
  9. Click the link "View in Google Play store", then click the button "Add to Wishlist"
  10. Login the "google play store" in both two test devices and choose Wishlist
  11. Install the application which you published
  12. Launch the application which you published

Expected Output:

  1. A dialog: "Add new application" is popuped
  2. The application is created successfully with the name which you inputed
  3. The apk for arm platform is uploaded into the application successfully
  4. Switch to advanced mode successfully
  5. The apk for x86 platform is uploaded into the application successfully
  6. Add Google Groups or Google+ Communities successfully
  7. The application is published successfully
  8. Become a tester for the application which you published successfully
  9. The application which you published is added to your Wishlist successfully
  10. Login the "google play store" and the wishlist is displayed successfully
  11. The application which you published is installed successfully
  12. The application which you published is launched successfully
