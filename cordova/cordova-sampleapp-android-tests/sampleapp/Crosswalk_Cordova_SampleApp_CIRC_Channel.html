Preconditions:

  1. Need a good network

Test steps:

  1. Connect to the IRC server for the first android device
    $ /server irc.freenode.net
  2. Set the nick name in the first android device, e.g. take "test_1" as the nick name
    $ /nick test_1
  3. Connect to the IRC server for the second android device
    $ /server irc.freenode.net
  4. Set the nick name in the second android device, e.g. take "test_2" as the nick name
    $ /nick test_2
  5. Use test_1 to join or create a channel, e.g. take "#channel_test" as the channel name
    $ /join channel_test
  6. Use test_1 to invite test_2 to join the channel
    $ /invite test_2 #channel_test
  7. Check if test_2 receive the "test_1 invites you to join #channel_test" message
  8. Use test_2 to join the "#channel_test" channel
    $ /join #channel_test
  9. Check if test_1 and test_2 are in the "#channel_test" channel member list
  10. Use test_1 to send a message to the "#channel_test" channel, take "hello" as the message
    $ /msg #channel_test hello
  11. Check if test_2 receive the "hello" message that test_1 send
  12. Use test_2 to send a message to the "#channel_test" channel, take "hello" as the message
    $ /msg #channel_test hello
  13. Check if test_1 receive the "hello" message that test_2 send
  14. test_1 leave the channel
    $ /part
  15. test_2 leave the channel
    $ /part
  16. test_1 disconnect to the IRC server
    $ /quit
  17. test_2 disconnect to the IRC server
    $ /quit

Expected Output:

  1. The first android device connect to the IRC server successfully
  2. The first android device set the nick name(e.g. "test_1") successfully
  3. The second android device connect to the IRC server successfully
  4. The second android device set the nick name(e.g. "test_2") successfully
  5. test_1 join or create a channel(e.g. "channel_test") successfully
  6. test_2 receive the "test_1 invites you to join #channel_test" message
  7. test_2 join the "channel_test" channel successfully
  8. test_1 and test_2 are in the "#channel_test" channel member list
  9. test_2 receive the "hello" message that test_1 send
  10. test_1 receive the "hello" message that test_2 send
  11. test_1 leave the "#channel_test" channel successfully
  12. test_2 leave the "#channel_test" channel successfully
  13. test_1 disconnect to the IRC server successfully
  14. test_2 disconnect to the IRC server successfully
