Precondition

  • Cordova Android 4+, CLI 5+
    $ sudo npm install cordova -g
  • minSdkVersion 16 (4.1 is the minimum supported version)
  • Set the environment path to the Android SDK Platform-tools* and **Android SDK Build-tools
  • Install Apache Ant
  • Set the JAVA_HOME to the environment path
Refer to https://github.com/mapsplugin/cordova-plugin-googlemaps/wiki/Tutorial-for-Mac

Test Steps

  1. Obtain the Google Maps API key for Android
    • Find the keytool
      ~/.android/
    • Display the SHA-1 fingerprint
      keytool -list -v -keystore ~/.android/debug.keystore -alias androiddebugkey -storepass android -keypass android
    • Go to https://code.google.com/apis/console/
    • Register your project
    • Turn on Google Maps Android API v2.
    • Go to API Access page.
    • Click [Create New Android Key] button
    • In the resulting dialog, enter the SHA-1 fingerprint, then a semicolon, then the package name: org.xwalk.hellomap
    • Write down the API Key See [the official document: Get an Android certificate and the Google Maps API key]
  2. Build
    • cd res/GoogleMapsPlugin
    • ./google-maps-plugin.py -k [google Maps API key]

    3. Details https://github.com/mapsplugin/cordova-plugin-googlemaps/wiki/Tutorial-for-CrossWalk-Webview-Plugin-%28Android%29

  3. Install and Run
    • The apk can be installed and uninstalled successfully
    • Launch the app, the google map load successfully
    • Click full screen, the google map should change to full screen, full screen button is hiddened
