Test that upgrade Crosswalk SDK version from V(n-1) to V(n) in embedded mode

Test steps:

  1. Build "demo" app with Crosswalk V(n-1):
    $ crosswalk-pkg --crosswalk=/path/to/crosswalk-[V(n-1)].zip --platform=android --android=embedded --targets=[arch] /path/to/testapps/demo
  2. Install Crosswalk runtime library V(n).
  3. Install the built "demo" with version "0.1", and launch it.
  4. Build "demo" app with Crosswalk V(n):
    $ crosswalk-pkg --crosswalk=/path/to/crosswalk-[V(n)].zip --platform=android --android=embedded --targets=[arch] /path/to/testapps/demo
  5. Install the built "demo" with version "0.1", and launch it.
  6. Uninstall the installed app
Note:
  1. Release candidate version of Crosswalk library = V(n)
  2. Released(old) app version = demo
  3. New app version = demo_upgrade
  4. These apks should be installed same test device

Expected Output:

  1. The apk named like "demo" with version "0.1" is built successfully.
  2. The Crosswalk runtime library is installed successfully.
  3. The app named like "Demo V0.1" is installed and launched successfully.
  4. The apk named like "demo" with version "0.1" is built successfully.
  5. The app named like "Demo V0.1" is installed and launched successfully.
  6. The app is uninstalled successfully.
