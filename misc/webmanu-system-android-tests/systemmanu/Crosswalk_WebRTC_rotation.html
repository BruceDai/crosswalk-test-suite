Precondition:

  1. Confirmed gravity sensing function is enabled.

Test steps:

  1. Use command "adb install webrtc.apk" to install webrtc.apk and open WebRTC app on device.
  2. Click "Start Connect" button.
  3. Write a message in input area and click "Send" button.
  4. Rotation device to change screen orientation.
  5. Write a message in input area and click "Send" button.

Expected Output:

  1. "pc1 and pc2 are connected successfully, and can send message now!" should display in the textarea above after clicking "Start Connect" button.
  2. Message should be sent successfully after clicking "Send" button.
  3. Message should be sent successfully after rotating device and clicking "Send" button.
