Precondition:

  1. Apple Mac OS X system
  2. Xcode 7.1+ installed
  3. Valid Apple Developer Account configured correctly in Xcode
  4. CocoaPods dependency manager for Xcode project installed

Test steps:

  1. Clone the crosswalk-samples repository:
    $ git clone https://github.com/crosswalk-project/crosswalk-samples.git
  2. Run the following commands to build the project:
    $ cd crosswalk-samples/fingerprint/iOS
    $ pod install
    $ open FingerprintDemo.xcworkspace
  3. Select the project "FingerprintDemo" --> set the active scheme --> FingerprintDemo --> <iOS Device> -- > Product --> Build

Expected Results:

  1. "Xcode" launches and a project named like "FingerprintDemo" is on the left
  2. Message "Build Succeeded" appears
