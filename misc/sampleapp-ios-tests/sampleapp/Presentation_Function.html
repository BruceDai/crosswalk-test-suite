Precondition

  1. Setup the test environment about how to make the test device can mirror onto the monitor screen, for example using MiBox and iPad mini 3
    1. Connect the MiBox to the display monitor (the HDMI converter may be needed here), and make sure the MiBox and test device are connected into the same WIFI network
    2. On iPad's home screen, swipe bottom-up to drag the action menu out. The "remote display" button should be appeared beside the "AirDrop" button. Select it and choose "MyBox(MiBox name)", and keep the "Mirroring" button enabled.Then the iPad's screen should be mirrored onto the display
  2. The test app named like: HexGLApp is available

Test steps:

  1. Install the app named like "HexGLApp" and launch it
  2. Change the "REMOTE DISPLAY:OFF" to "REMOTE DISPLAY:ON" on the lauching screen
  3. Click "Start"
  4. Touch the screen
  5. Play the game until display "DESTROYED"
  6. Touch the screen to restart the game

Expected Results:

  1. Launch the app named like "HexGLApp" successfully, and no error message, no crash, and display "START", "CONTAOLS", "REMOTE DISPLAY:OFF" and so on
  2. Display "REMOTE DISPLAY:ON"
  3. Display "CLICK/TOUCH TO CONTINUE", "TURN" and "ACCEL"
  4. Display the game main page normally on the external display and enter the "START" page and you can play the game when display "GO" on the screen
  5. Display message like: "CLICK/TOUCH TO CONTINUE" and play time
  6. Display "START", "CONTAOLS", "REMOTE DISPLAY:OFF" and so on
