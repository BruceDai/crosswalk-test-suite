Pre-condition

Before run the FingerprintDemo, please first set a touch ID lock for an iOS device.

Test steps:

  1. Build the project, please refer to Build FingerprintDemo project
  2. Connect the iOS device to the Mac OS X system. In Xcode, select Product --> Run to launch the app on the iOS device
  3. Check the UI of app FingerprintDemo
  4. Click Authenticate with fingerprint button
  5. Put your finger on the touch ID sensor of the iOS device

Expected Results:

  1. Message Build Succeeded displays on Xcode
  2. An app named FingerprintDemo launches successfully, and no error message, no crash
  3. There is a button Authenticate with fingerprint on the app UI
  4. A dialog popups to ask your put your finger on the touch ID sensor
  5. The dialog disappears and a new line text Autentication passed! appears below the button.
