# Native Client Test Suite

## Introduction

This test suite is for checking the Native Client (NaCl) support in
Crosswalk Project:
* https://crosswalk-project.org/jira/browse/XWALK-1214

## Pre-conditions

Make sure there is a NaCl server connected.
